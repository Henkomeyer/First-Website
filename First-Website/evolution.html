The first implementation of the web and lasted from 1989 to 2005. Also defined as the web of information connections. Tim Berners Lee considered this web as the “Read only web”, this was because of very little interaction with the website. This first-generation web had static pages and was used for content delivery purposes only, this meant that you could search for information and display it but there was no user interaction (Gregersen, 2019).

Web 1.0

The web has evolved a lot from what it was when it first started, from where Tim Burners invented the web to where it is today. Tim Burners lee viewed the capabilities of the web, by three expressions or phases rather, firstly he named the web of containing documents (Web 1.0) then the web of interactive People (Web 2.0) and the Web of organised Data (Web 3.0).

By this the web has been through several changes in its development cycle and going by the current thread of evolution the web is shifting to a more data focused phase (Choudhury, 2014).

Characteristics Limitations
A system of connected computer networks, making use of IP protocols to link devices. Read only content
Static web pages that used basic HTML Web masters are solely responsible for updating and managing the content of the web pages.








































Web 2.0

Set alight in 2004 by Dale Dougherty, this web is also known as the wisdom web or read-write web, only naming a few. This web would be what is the first bi-directional web with reading as well as writing capabilities, in other words the users of web 2.0 have more interaction with less control in things like a flexible web design, collaborative content creation and the most important to support collaboration and gaining collective intelligence (Sareh Aghaei, 2012).

Characteristics Limitations
Web is a platform with software greater than a single device, technology like blogs, wikis, podcasts, RSS feeds etc. Iteration’s cycle of change and constant updates to service
Business revolution as the computer industry caused a step in using the internet as platform. Ethical issues concerning the build and usage of the web
Content management and new ways of interaction between users, social media etc, that increases user to user information sharing. Knowledge sharing and connection between peers are still limited.

Web 3.0

First observed and thought out by John Markoff of the New York times in 2006, also called the executable web. The idea of the third-generation web was to define and link structure data to gain more effective discoveries, make automation integration and make it accessible across various applications (Max Mersch, 2019).

Web 3.0 is referred to as the semantic web, with a purpose of making the evolution of the current web for users to locate and interact information more fluently. The overall purpose of a semantic web is to give machines the ability to understand and respond to human requests, this meant that the information had to be ordered in a semantical structure so that the device could use it (Choudhury, 2014).

Characteristics Limitations
There is an open-source software platform. Redundancy in data as there is a lot of data that is processed at the same time.
The distribution of the worldwide database. A local problem of deceit where a producer of information is intentionally sharing misleading information.
Abilities such as web personalisation, resource pooling and an overall intelligent web. Inconsistency caused by logical contradictions that will rise during the development of large sections of data.
